ProxiFarm

ProxiFarm est une application mobile en Flutter créée pour aider les producteurs locaux à vendre leurs produits en ligne, tout en promouvant l'achat local. Développée en 6 mois en collaboration avec un ami.

Technologies : Flutter, Firebase, Dart, JavaScript, GitHub.

Fonctionnalités : Ajout d'annonces, gestion de comptes, paiement, discussions, notifications, statistiques.

CivilyNation

CivilyNation est un plugin Minecraft développé pour simuler l'économie et la gestion d'un pays dans un environnement multijoueur. Créé pour offrir une nouvelle expérience de jeu afin de s'amuser entre amis.

Technologies : Java, Minecraft API.

Fonctionnalités : Gestion de pays, commerce, économie, administration.

ProxiFarm - Site Web

Un site web pour promouvoir l'application ProxiFarm et permettre aux utilisateurs d'accéder à des tutoriels sur l'utilisation de l'application, trouver les liens de téléchargements ou encore accéder à la page afin de bénéficier de code cadeau. Dans un premier temps, la page web permettait de souscrire à des abonnements pour en bénéficier sur l'application mobile.

Technologies : HTML, CSS, JavaScript, PHP, Stripe API.

Fonctionnalités : Tutoriels, Authentification, Gestion de comptes, Gestion de paiement via Stripe
